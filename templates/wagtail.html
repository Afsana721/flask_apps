How data flows

1. User writes in the editor or enters a prompt.
2. Frontend sends the text to a Django API route such as /api/ai.
3. Django validates the request and calls AI or search services.
4. Response returns as JSON with content, validation, and sources.
5. The UI updates output cards and source blocks automatically.
